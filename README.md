# Optimizing Toric Code Decoding with Deep Reinforcement Learning

[colab link here]

This repository follows the framework outlined in **arXiv:1811.12338**, utilizing Deep Reinforcement Learning (DRL) to optimize error correction pathways on the Toric Code.

Using **PyTorch**, this project builds and trains a Convolutional Neural Network (CNN) to model the action-value ($Q$) function of an RL loop. The agent is trained to systematically navigate and annihilate anyon strings generated by dephasing noise ($p=0.4$). The resulting lightweight decoder achieves **70% logical fidelity** with an aggressive training footprint of only five minutes.

---

## Architecture & Core Components

* **Defect-Centered Toric Code Environment:** Implements the geometry necessary to put the Toric Code on a 2D torus. It handles error generation, syndrome extraction, state representation, and reward shaping tailored for the reinforcement learning loop.
* **Toric Anyon Deep Q-Network (DQN):** Defines the deep convolutional network used to approximate $Q$. Crucially, it incorporates a custom **circular padding layer** to enforce the periodic boundary conditions of the torus.
* **Optimized Training Loop:** Employs Deep Q-Learning with the Adam optimizer, integrated with a dynamic learning rate scheduler and an $\epsilon$-greedy exploration policy. This configuration stabilizes convergence against the higher learning rates required for fast string-matching progression.
* **Decoder Evaluation Toolkit:** Extracts the optimal model weights from the training run to evaluate the logical error rate over a broad sample distribution, calculating the fidelity metrics quoted above.
